4d-plugin-xslt
==============

4D implementation of [libxslt](http://xmlsoft.org/libxslt/). 64 bits OS X and Windows

Previous (v16 or earlier) available in [thread-unsafe](https://github.com/miyako/4d-plugin-xslt/tree/thread-unsafe) branch
